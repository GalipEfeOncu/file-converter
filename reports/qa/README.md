# QA Reports

This directory contains regression test logs and quality assurance documentation.

## Guidelines
- Sprint QA logs should be saved as `sprint_N_qa.md`.
- The main test matrix is tracked in `E2E_TEST_MATRIX.md`.
- All manual testing results must be documented here before a release.

## E2E Test Matrix Format
Markdown table with columns: 
`Feature | File Type | Expected | Result | Notes`
